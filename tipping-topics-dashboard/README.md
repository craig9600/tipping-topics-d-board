# Tipping Topics Dashboard

A clean, branded dashboard showing your betting tips performance.

## Setup Instructions

### 1. Install Node.js
Download and install from https://nodejs.org (LTS version)

### 2. Create GitHub Account
Go to https://github.com and sign up (free)

### 3. Create a New Repository
- Click "New repository"
- Name it `tipping-topics-dashboard`
- Make it **Public**
- Click "Create repository"

### 4. Upload Files
Upload all files from this folder to your new repository:
- Either drag & drop in the GitHub web interface
- Or use GitHub Desktop (easier for beginners)

### 5. Deploy to Vercel
1. Go to https://vercel.com
2. Sign up with your GitHub account
3. Click "New Project"
4. Import your `tipping-topics-dashboard` repository
5. Click "Deploy"
6. Wait 1-2 minutes and you'll get a free URL like `tipping-topics-dashboard.vercel.app`

### 6. Update Your Data

Every time you want to update the dashboard:

1. Run `export_dashboard_data.py` on your PC (this creates the JSON)
2. Copy the updated `dashboard_data.json` to the `public/data/` folder
3. Push to GitHub (Vercel auto-deploys)

#### Automated Option (Windows Task Scheduler)
You can create a batch file to run after your bet finder:

```batch
@echo off
python value_bet_finder_v2.py
python export_dashboard_data.py
cd C:\Users\craig\OneDrive\Desktop\tipping-topics-dashboard
git add .
git commit -m "Update data"
git push
```

## Customization

### Colors
Edit the `COLORS` object in `src/App.jsx`:
```javascript
const COLORS = {
  navy: '#1e3a5f',
  gold: '#f4b942',
  // etc
};
```

### Content
All data is read from `public/data/dashboard_data.json` which is generated by `export_dashboard_data.py`.
